# 图片生成提示词参考

常用的 SeeDreamV5 图片生成提示词示例，可直接复制使用。

```
一只可爱的卡通小猫正在玩毛线球，明亮的色彩，3D渲染风格，图片风格为：cute
```

```
一只圆滚滚的柴犬坐在草地上，头上戴着花环，Q版风格，柔和光线，图片风格为：cute
```

```
一个小女孩抱着一只泰迪熊，背景是星空，插画风格，温馨配色，图片风格为：cute
```

## 小红书配图常用

```
可爱风格插画，关于AI工具推荐，小红书风格，1:1比例，明亮色彩，Q版，无文字
```

```
美食摆盘俯拍，精致甜点搭配鲜花装饰，小红书风格，高饱和度，清新配色
```

## 提示词写作技巧

1. **描述主体**：明确说明画面中的主要元素
2. **指定风格**：在提示词末尾添加 `图片风格为：xxx` 来控制整体风格
3. **控制构图**：使用 "特写"、"俯拍"、"全景" 等构图词
4. **调整色调**：使用 "暖色调"、"冷色调"、"高饱和度" 等色彩词
5. **排除文字**：如果不需要文字，加上 "无文字" 或 "不含文字"
6. **宽高比配合**：
   - `1:1` — 小红书配图、头像
   - `9:16` — 手机壁纸、竖屏海报
   - `16:9` — 公众号封面、横屏展示
